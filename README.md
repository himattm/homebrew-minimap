# Himattm Minimap

## How do I install these formulae?

`brew install himattm/minimap/<formula>`

Or `brew tap himattm/minimap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "himattm/minimap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
